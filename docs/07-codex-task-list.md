# 07 — Lista de Tareas para Codex

Codex debe trabajar en tareas pequeñas y verificables.

- Crear proyecto Next.js.
- Instalar Tailwind/shadcn.
- Configurar Prisma.
- Crear schema inicial.
- Crear seed.
- Crear Home.

## Sprint 2 — Ranked básico

- Crear endpoint start match.
- Crear endpoint answer.
- Crear endpoint finish.
- Crear pantalla match.
- Registrar respuestas.
- Mostrar resultado.

## Sprint 3 — Rango y LP

- Crear sistema de MMR.
- Crear sistema de LP.
- Implementar rangos.
- Agregar tests.

## Sprint 4 — Banco sin repetición

- Crear `Fact`.

- Crear historial.
- Implementar selector anti-repetición.
- Agregar tests.

- Ocultar under_review del ranking.

## Sprint 6 — Leaderboard
